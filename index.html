Tic-Tac-Toe Game
front-end  UI design
HTML 5, CSS, Java Script

This project was set up the 3x3 grid with squares positioned on the page and uses two different images for players to move around the squares. When is not the player’s turn, they cannot move their pieces, also once an image has been dropped into a square, it cannot be moved again. Pair Project.
